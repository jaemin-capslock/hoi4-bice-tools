BICE Division Stats Calculator

A Python toolkit for computing and comparing division stats from the Black ICE mod for Hearts of Iron IV. Parses Clausewitz scripts directly — always in sync with your installed mod.

1 500+ Equipment 300+ Battalions 60+ Tank Designs 123+ Tests

Features

⚙

Clausewitz Parser

Parses HOI4 mod files directly. Equipment DB (1 500+ entries) and battalion DB (300+ types) built from source — always accurate to your mod version.

⚔

Division Calculator

Full division stat aggregation matching HOI4 formulas. Supports line battalions, support companies, and category-aware doctrine modifiers.

🛠

Tank Designer (TRM)

Modular Tank Rework system with 1 136 modules. 60+ pre-built German historical designs from PzKpfw I to Maus.

📊

Flask Web UI

Interactive browser with equipment/battalion tables, real-time division designer with drag-and-drop, stat computation, and icon rendering.

📑

Excel Export

Generate styled workbooks with division comparisons, equipment references, and battalion data with color-coded heat maps.

🛠

Doctrine System

WW1 doctrine presets with 55 cumulative techs. Category-aware multiplicative/additive stacking matching HOI4 internals.

API Usage

Per-Battalion Stats

from bice_parser import build_equipment_db, build_battalion_db
from bice_calc import calc_battalion

equip_db = build_equipment_db()
bat_db   = build_battalion_db()

stats = calc_battalion(
    "infantry",
    ["infantry_equipment_2", "infantry_uniforms_2"],
    bat_db, equip_db,
)
# → {"soft_attack": 2.1, "defense": 6.4, "breakthrough": 0.9, ...}

Division Designer

from bice_models import BICEDatabase, Division
from bice_doctrines import get_preset

db = BICEDatabase()

div = (Division("9 Inf / 4 Art", db)
    .add_battalion(db.battalion("infantry").equip_auto(1939), 9)
    .add_battalion(db.battalion("artillery_brigade").equip_auto(1939), 4)
    .add_support(db.battalion("DIV_HQ").equip("infantry_uniforms_2"))
    .add_support(db.battalion("engineer").equip_auto(1939)))

stats = div.compute(get_preset("ww1_full"))

Tank Designer

from bice_tanks import build_module_db, inject_tank_designs, GERMAN_TANK_DESIGNS

module_db = build_module_db()   # 1 136 tank modules

for d in GERMAN_TANK_DESIGNS:
    stats = d.compute_stats(equip_db, module_db)
    print(f"{d.name}: SA={stats['soft_attack']:.1f}")

# Inject into equipment DB for division use
inject_tank_designs(equip_db, module_db=module_db)

Division Stat Aggregation

These rules match HOI4's internal formulas:

StatRule
HP, Soft Attack, Hard Attack, Air Attack, IC CostSum across all battalions (count × stat)
Organization, Defense, Breakthrough, HardnessWeighted average by battalion manpower
SpeedMinimum across non-support transport equipment
Combat WidthSum (support companies contribute 0)
Training TimeMaximum across all battalions
